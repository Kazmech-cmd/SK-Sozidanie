# СК Созидание — Веб-сайт строительной компании с интерактивным подбором недвижимости

Современный и адаптивный корпоративный веб-сайт для строительной компании «СК Созидание», специализирующейся на жилом и коммерческом строительстве. Проект включает в себя динамический каталог объектов, интерактивную систему фильтрации квартир по параметрам на стороне сервера и автоматическую интеграцию с мессенджером Telegram для мгновенного оповещения менеджеров о новых заявках.

Основной функционал

Динамический каталог недвижимости: Автоматический вывод карточек строящихся объектов и доступных квартир из базы данных MySQL.
Умный многопараметрический фильтр: Позволяет пользователям интерактивно подбирать жилье по количеству комнат, диапазону площади (м²), стоимости (₽), этажу, а также выполнять текстовый поиск по названию ЖК и сортировать результаты (сначала дешевле/дороже).
Автоматизация лидогенерации: Интегрированная форма обратной связи с валидацией данных. При отправке заявки клиентом, менеджер мгновенно получает детализированное уведомление в Telegram-канал через Bot API.
Двухсредовая конфигурация (Роутинг БД): Конфигурационный файл базы данных автоматически определяет окружение (`localhost` или удаленный хостинг) и переключает доступы, что обеспечивает бесшовную разработку без необходимости постоянной правки кода.
Высокая адаптивность: Интерфейс полностью оптимизирован под любые типы устройств (десктопы, планшеты, смартфоны).

Технологический стек

Front-end: HTML5, CSS3 (Flexbox, Grid, CSS Variables), JavaScript (ES6+, Fetch API для асинхронной отправки форм).
Back-end: PHP 8.x (обработка параметров фильтрации, динамическая сборка страниц через `include`/`require`).
База данных: MySQL / MariaDB (работа через расширение `mysqli`, оптимизированные SQL-запросы выборки).
Интеграции: Telegram Bot API.
